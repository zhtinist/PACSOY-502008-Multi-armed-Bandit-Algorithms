关于乐观初始化：
将prior_means初始化为最大值B（prior_means = np.ones(K) * B），而非原来的0。
这使得算法在早期更倾向于探索所有臂（因为初始采样值较高），从而加速收敛。
效果：
早期探索更充分，避免陷入局部最优。
累积遗憾曲线上升速度更快趋于平缓（更快找到最优臂）。
注意事项：
乐观初始化适用于奖励范围已知的场景（此处B已知）。
如果奖励范围未知，可能需要其他探索策略（如自适应初始化）。
